What is this project

This is a public fork of Google's ExoPlayer. In this project, we modify the demo and ExoPlayer itself to support following

  • Keyboard shortcuts for controlling all playback actions
  • Keyboard shortcuts for toggling all indicators/lables on the screen
  • A-B repeating
  • Zoom and Pan
  • Synchronized playback amoung multiple devices
  • Read content list from json file
  • Playback from local storage

ExoPlayer

ExoPlayer is an application level media player made by Google. Please refer to the original README.MD of ExoPlayer for more detail.

Project branches

  • ab-dev-vX is the development branch of this project, where X is the major version number of ExoPlayer.
  • Pull requests should normally be made to this branch.
